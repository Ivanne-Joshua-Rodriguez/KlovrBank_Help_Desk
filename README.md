# KlovrBank_Help_Desk
A help desk for a fictional digital bank, featuring account handling, an AI chatbot through the Botpress API, and live communication between users and support, and support and admin, along with other features.
This is a project done for the requirements for the Software Design subject.
The website is deployed digitally through InfinityFree and can be accessed through this link (https://klovrbank.gt.tc/)
The codes in this repository are for local deployment only.
